# Docker Assignment

## Project
Simple Node.js app running inside Docker container.

## Steps Performed
- Created Node.js app
- Wrote Dockerfile
- Built Docker image
- Ran container
- Pushed image to Docker Hub

## Docker Hub Link
https://hub.docker.com/r/kpragati03/docker-assignment

## How to Run
docker pull kpragati03/docker-assignment
docker run -p 3000:3000 kpragati03/docker-assignment
